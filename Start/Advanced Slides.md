# Präsentation starten

Zum Starten gibt es mehrere Optionen:

1. Icon links klicken mit dem Monitor mit Inhalt
2. Command pallette:
	 `cmd` (`strg`) + `p`  + `Advanced Slides: Show Slide Preview`
